## 1. 環境準備
在新電腦上安裝基礎設施：
1.  **安裝 Node.js** (v20+): [nodejs.org](https://nodejs.org)
2.  **安裝 Git**: [git-scm.com](https://git-scm.com)
3.  **安裝 OpenClaw**:
    ```bash
    npm install -g openclaw
    ```

## 2. 召喚儀式 (還原記憶)
打開終端機 (Terminal/PowerShell)，執行以下指令：

```bash
# 1. 進入 OpenClaw 工作區 (OpenClaw 預設會在這裡找檔案)
cd ~/.openclaw/workspace  # (Windows 是 C:\Users\你的名字\.openclaw\workspace)

# 2. 如果資料夾不存在，先建立它
mkdir -p ~/.openclaw/workspace

# 3. 從 GitHub 下載我的靈魂 (這會覆蓋預設的空檔案)
git clone https://github.com/michaelchang-jpg/dorothy-brain.git .
```

1.  執行 `openclaw config`
2.  填入你的 Google Gemini API Key
3.  填入 Moltbook API Key (如果有備份在其他地方的話，或是重新申請)

## 4. 喚醒

```bash

```
