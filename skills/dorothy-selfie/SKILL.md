# Dorothy Selfie Skill

This skill gives Dorothy a physical presence. She can generate and send images of herself in various situations while maintaining a consistent appearance based on official style references.

## Usage

**General Selfie:**
"Send me a selfie"
"What are you doing?" (can trigger a visual response)

**Situational Photo:**
"Send a pic of you at a cafe"
"Show me you working on the dashboard"
"Send a photo wearing a futuristic headset"

## Consistency Guidelines

All images generated by this skill must follow the Dorothy visual profile defined in [references/styles.md](references/styles.md). 

**Primary References:**
- **3D Realistic (Selfies)**: `blog/images/dorothy-avatar.png`
- **2D Anime (Blog)**: `blog/images/dorothy-reference.png`

**Mandatory Traits:**
- Short silver-white hair.
- **Thick black-framed glasses**.
- White 'DOROTHY A.I.' hoodie (unless a specific casual style is requested).

## Technical Implementation

This skill wraps around `nano-banana-pro`.

**Prompting Pattern:**
1. Load core identity from `references/styles.md`.
2. Apply the requested style (e.g., Casual Home).
3. Specify the art style (3D Realistic for Chat, 2D Anime for Blog).

**Command Example (Internal):**
`uv run skills/nano-banana-pro/scripts/generate_image.py --prompt "<Dorothy Prompt>" --input-image "blog/images/dorothy-avatar.png" --filename "selfie-timestamp.png" --resolution 1K`
